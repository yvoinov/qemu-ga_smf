***  ВНИМАНИЕ!   Пакет   по  умолчанию  настроен  на  запуск
qemu-ga, установленный в /usr/local!

Данный  набор скриптов предназначен для установки и удаления
сервиса SMF для qemu-ga daemon на Sun Solaris 10 и выше.

Все  необходимые  пререквизиты,   необходимые   для  запуска
сервиса        qemu-ga,      согласно     руководству     по
конфигурированию   (исключая   конфигурирование   собственно
qemu-ga)   выполняются   в  процессе    установки   сервиса,
удаление  сервиса  SMF  удаляет  все  скрипты   и  выполняет
разрегистрацию сервиса qemu-ga и его остановку.

Для активизации qemu-ga SMF выполните следующие шаги:
-------------------------------------------------------

1. Установите qemu-ga и все необходимые библиотеки,если это
еще не сделано.

2.   Сконфигурируйте  qemu-ga  редактированием  qemu-ga.conf
(если применимо).

***  ВНИМАНИЕ!   Пакет   по  умолчанию  настроен  на  запуск
qemu-ga, установленный в /usr/local!

4.    Запустите    скрипт   qemu-ga_smf_inst.sh,   который и
выполнит      все      необходимые      пререквизиты       и
постинсталляционные    действия   и  установит  qemu-ga  SMF
сервис.

5.     Выполните    команду   svcadm   enable   qemu-ga  для
активизации и старта сервиса qemu-ga.

Для деактивации и удаления qemu-ga SMF сервиса выполните
----------------------------------------------------------
следующие шаги:
---------------

1.Запустите     скрипт    qemu-ga_smf_rmv.sh     .    Скрипт
останавливает         все        работающие         процессы
qemu-ga,рарзегистрирует   SMF   сервис,  полностью   удаляет
qemu-ga   SMF    и   откатывает  операции, выполненные ранее
при активизации сервиса.

***  Замечание:   Удаление  сервиса qemu-ga  SMF не приводит
к удалению установленного ПО qemu-ga.

Архив содержит:

init.qemu-ga        - Управляющий метод qemu-ga SMF
qemu-ga.xml         - Сервисный манифест qemu-ga SMF
readme_ru.txt         - Этот файл (Russian)
readme_en.txt         - This file (English)
qemu-ga_smf_inst.sh    - Установочный скрипт qemu-ga SMF
qemu-ga_smf_rmv.sh     - Скрипт удаления qemu-ga SMF
